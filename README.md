
AR相机
===================================
学习GLSL语言的练习项目。
  
在大神的开源项目基础上扩展了一些滤镜。
  
比较能拿得出手的是吐彩虹滤镜。
  
Base on 
----------------------------------- 
* [AndroidOpenGLDemo](https://github.com/doggycoder/AndroidOpenGLDemo) by doggycoder
* [Omoshiroi](https://github.com/Martin20150405/Omoshiroi) by Martin20150405
